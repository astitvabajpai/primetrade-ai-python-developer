# Hyperliquid × Fear & Greed — Trading Intelligence

> Exploring the relationship between Bitcoin market sentiment and trader performance on Hyperliquid DEX.  
> Built for the Python Development Intern assignment 

---

## Overview

This project analyzes **211,000+ trades** from 32 Hyperliquid wallets alongside the **Bitcoin Fear & Greed Index** (2018–2025) to uncover hidden patterns between market sentiment and trader profitability.

**Key questions answered:**
- Does market sentiment (Fear vs Greed) predict trader profitability?
- Do long and short traders perform differently across sentiment zones?
- Which coins are best traded during specific sentiment conditions?
- What time of day yields the highest average PnL?
- What separates top traders from average ones?

## Setup & Usage

### 1. Install dependencies

```bash
pip install pandas numpy scipy
```

### 2. Run the analysis

```bash
python analysis.py
```

This will:
- Merge both datasets on date
- Run 10 analysis modules (see below)
- Print a full **Key Insights Summary** to the terminal
- Save all results to `output/` as CSVs
- Generate `insights.json` for the dashboard

### 3. Open the dashboard

Simply open `dashboard.html` in any browser:

```bash
# macOS
open dashboard.html

# Windows
start dashboard.html

# Linux
xdg-open dashboard.html
```

> No server required. The dashboard runs fully offline and reads `insights.json` automatically.

## Key Findings

### Sentiment & PnL
| Sentiment | Avg PnL/trade | Win Rate | Total PnL |
|---|---|---|---|
| Extreme Fear | $34.5 | 37.1% | $0.74M |
| Fear | $54.3 | 42.1% | $3.36M |
| Neutral | $34.3 | 39.7% | $1.29M |
| Greed | $42.7 | 38.5% | $2.15M |
| **Extreme Greed** | **$67.9** | **46.5%** | **$2.72M** |

**Extreme Greed** is the most profitable sentiment zone — the highest avg PnL, highest win rate, and strong total profits.

### Long vs Short
- Short trades **outperform longs in every single sentiment zone**
- Most extreme gap during **Extreme Greed**: shorts avg $114.6 vs longs at $10.5 — a **10× difference**

### Coin-Specific Patterns
| Coin | Best Sentiment | Avg PnL |
|---|---|---|
| ETH | Extreme Fear | $196.9 |
| MELANIA | Extreme Fear | $218.3 |
| SOL | Greed | $284.8 |
| @107 | Extreme Greed | $191.2 |
| BTC | Fear | $48.9 |

### Time of Day (IST)
- **12:00** is the single best hour — avg **$131/trade** (3× the daily average)
- 07:00 and 11:00 also consistently outperform
- Weakest hours: 23:00 ($18.8) and 14:00 ($23.1)

### FG Index ↔ PnL Correlation
- Pearson r = **0.037** — essentially zero
- Sentiment is useful as a **directional filter**, not a return predictor

### Top Trader Insight
- Best wallet: **$2.14M total PnL** with only **34% win rate** across 14,700 trades
- This points to an **asymmetric strategy** — small losses, large wins — not grinding consistency
